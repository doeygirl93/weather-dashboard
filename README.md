# project-weather-dashboard
## Desc
This is a Weather dashboard, it has a landing page, auth/login page and the acutal dashboard. It displays your local time, weather and provides the photo of the day from nasa + it has inspirational quotes to motivate you

## Features
It has a Landing page with varies photos from nasa
Dragable stickers int eh Home page
An About section where I yap
A cute login page to enter your name
Nasa photo of the day on the dashboard
Displays User's time
Displays User's local weather
Displays random inspirational quote

## How I made it
I made it through using the Resolution week 3 tutorial as the base/guideline. 
My stack is Sveltekit, Typescript (kinda?), Tailwind Css, Svelte Motion (motion.dev for svelte), Unsplashed (for images), & npm
